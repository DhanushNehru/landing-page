# Landing Page

This is a landing page template to customize and use if for your own needs.

A Vite + React + Tailwind TypeScript landing page template built with shadcn/ui primitives and Radix components.

Requirements
- Node.js (>=16 recommended)
- npm or pnpm
- Development container: Ubuntu 24.04.2 LTS (this workspace runs in a dev container)

Development

Install dependencies
```bash
npm install
```

Run the dev server (hot reload)
```bash
npm run dev
```
